Deployment

To help us get our site live we will be using surge.sh Surge is a static web publishing tool that makes deploying websites quick and easy. There are limitations but it is fine for our purposes. Surge has an excellent video that explains how to deploy from the command line. I recommend you watch it

INSTRUCTIONS

  1. ensure you have a recent version of Node.js
  2. Then from the comannd line install Surge using npm by running the following command: `npm install --global surge`. You may need to preface this command with sudo
  3. If you make an edit to your site and want to redeploy `surge --domain chloedonegan.surge.sh`
  4. Wouldn't it be nicer just to be able to type `surge`? Save your domain name to a cname file - in the command line 'echo chloedonegan.surge.sh > CNAME'
  5. Make a change , then type surge into your comand line and check out your site again!
